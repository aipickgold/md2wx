## 🚀 Killer Feature: One-click Publish to Draft

Other tools let you "copy → switch to WeChat backend → paste → confirm style". **md2wx pushes directly to your draft box**:

```
Markdown → 🚀 One-click Publish → WeChat Draft Box
```

- Bring your own AppID + AppSecret (stored only in your browser)
- Hit "Publish" → 3 seconds to draft
- Pro feature · the only real differentiator among Chinese Markdown editors

## 🚀 Quick Start

### Browser (recommended)

```
1. Open https://aipickgold.com
2. Paste Markdown into the left editor
3. Pick a theme → click "Copy" (top-right)
4. Paste into WeChat Official Account backend → done!
```

### API (Pro)

```bash
curl -X POST https://aipickgold.com/api/convert \
  -H "X-API-Key: your-key" \
  -d '{"markdown": "# Hello", "theme": "bytedance"}'
```
