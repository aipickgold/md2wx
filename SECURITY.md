# Security Policy

## 📬 Reporting a Vulnerability

如果你发现了安全漏洞,请 **不要公开提 Issue**。请通过以下方式私下告知我们:

- **邮箱**:`soarwill06@gmail.com`
- **主题**:`[SECURITY] md2wx - 简要描述`
- **内容**:
  - 漏洞描述与影响范围
  - 复现步骤(越详细越好)
  - 受影响的版本
  - 如可能,建议的修复方案
  - 你希望的披露时间表

我们承诺:

- **48 小时内** 回复确认收到
- **7 天内** 给出初步评估和修复计划
- 漏洞修复后,会在 `CHANGELOG.md` 致谢(除非你希望匿名)

## 🔑 PGP
