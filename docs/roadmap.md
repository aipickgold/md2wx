# md2wx · Roadmap

> 规划可能随社区反馈调整。有建议欢迎提 [Issue](https://github.com/aipickgold/md2wx/issues)。

## 近期(6 周内)

- [ ] **截图 / demo GIF 上传** — README + docs 里的 "截图建设中" 占位替换
- [ ] **主题画廊 PR 开放** — 社区可贡献新主题,通过 PR 审核合入
- [ ] **v1.1.0 Release** — 打包当前稳定版本 + 发布说明
- [x] **一键发布到公众号草稿箱**(Pro) — 2026-04 上线,BYO AppID+Secret 模式

## 中期(6 个月内)

- [ ] **AI 写作助手集成** — 与 Claude Code / OpenClaw 协同,"写完即排版"
- [ ] **自定义主题编辑器** — 不改代码也能调颜色/字体生成新主题
- [ ] **多公众号账号切换** — 高级用户可同时管理多个公众号
- [ ] **批量草稿上传** — 一次提交多篇文章到草稿箱
- [ ] **英文化 UI** — 面向海外创作者

## 长期(2026-2027)

- [ ] **自托管 Docker 镜像** — 企业/团队私有部署
- [ ] **团队协作版** — 多作者编辑、审阅、发布流程
- [ ] **公众号以外的发布目标** — 头条号、企鹅号、百家号、知乎
- [ ] **Markdown Linter** — 实时检测公众号兼容性问题
