# md2wx · 常见问题

完整版请访问 [aipickgold.com/help](https://aipickgold.com/help)。

## 基础使用

**Q: 怎么开始?**
A: 打开 [aipickgold.com](https://aipickgold.com),粘贴 Markdown,选主题,点右上角「复制」,到公众号后台粘贴即可。

**Q: 支持哪些 Markdown 语法?**
A: 完整 GFM。H1-H6 / 加粗 / 斜体 / 链接 / 图片 / 列表 / 表格 / 代码块(含语法高亮) / 引用 / 分割线 / 脚注 / 提示框 / 对话框 / 图片画廊 / 长图容器。

**Q: 复制到微信后样式丢了?**
A: 必须用编辑器的「复制」按钮(不是 Ctrl+C),直接粘到公众号后台,**不要**经过 Word / 记事本中转。

## 一键发布(Pro 独家)

**Q: 一键发布和复制粘贴有什么区别?**
A: 复制粘贴要 4 步(复制 → 切后台 → 新建图文 → 粘贴),一键发布 1 步直达草稿箱。需要 AppID + AppSecret。

**Q: 需要什么样的公众号?**
A: 已认证的订阅号或服务号。个人订阅号因为微信 API 限制暂不支持。

**Q: 凭证安全吗?**
A: 只保存在你的浏览器 localStorage,不上传我们服务器。随时可在设置里「断开」彻底清除。

## 主题

**Q: 40 个主题怎么挑?**
A: 经典(通用)/ 极简(深度文章)/ 聚焦(正式)/ 渐变(品牌)/ 卡片(产品)。每系列 8 种配色,挑顺眼的。

## 付费

**Q: 免费版能用吗?**
A: 可以。10 基础主题 + 完整功能(带水印)。重度创作者建议 Pro。

**Q: 合并 ¥299 比单品 ¥199 贵多少?**
A: 只多 ¥100,但拿两个产品永久。如果你既写公众号又做小红书,这是最划算的。

## 其他

**Q: 浏览器要求?**
A: Chrome / Edge / Safari / Firefox 最新版。推荐 Chrome(剪贴板兼容性最好)。

**Q: 移动端能用吗?**
A: 编辑器针对桌面优化。移动端可以浏览但编辑体验一般。
