# md2wx · 10 个使用场景示例

每个示例:用户输入 + 预期 Skill 行为。

## 1. 产品发布文

```
用户:帮我写一篇介绍新产品 XYZ 的公众号,品牌风格,突出 3 个卖点

Skill:
  - 主题推荐:gradient-blue 或 bytedance-orange
  - 结构:Hero 大标题 → 3 小标题(每卖点一节) → CTA
  - 起草 → 渲染 → (可选)推送草稿
```

## 2. 技术深度文章

```
用户:把这段 markdown 排成公众号格式,用极简黑主题

Skill:
  - 主题:minimal-black
  - 保留代码块高亮
  - H1-H6 精致递进
  - 推送草稿
```

## 3. 财经评论

```
用户:写一篇关于 A 股新政的评论,2000 字,聚焦蓝主题

Skill:
  - 主题:focus-blue
  - 结构:现象 → 分析 → 预测 → 建议
  - 加引用块(观点)+ 加粗关键数据
```

## 4. 金句短内容

```
用户:出一篇 300 字短评,主题是 AI 工具对创作者的意义

Skill:
  - 主题:gradient-purple(短内容更抢眼)
  - 3-5 段落,每段 100 字内
  - 直接复制到公众号
```

## 5. 访谈整理

```
用户:把这段录音转写整理成公众号访谈文章

Skill:
  - 用 ::: dialogue 对话框语法
  - 问答交替,自动分配彩色头像
  - 主题建议:卡片系列(分块清晰)
```

## 6. 教程指南

```
用户:写一个 5 步教程,教用户如何用 ChatGPT 写代码

Skill:
  - 主题:focus-green(步骤感)
  - 每步独立小节 + 代码块示例
  - H2 序号 "第一步 / 第二步" 自动高亮
```

## 7. 周报 / 月度总结

```
用户:整理我这个月写的 10 篇文章的精华,做成一期月报

Skill:
  - 主题:classic-brown(温度)
  - 有序列表罗列 + 每条简评
  - 结尾加订阅引导
```

## 8. 活动公告

```
用户:下周我要办一场线下分享,写一篇报名通告

Skill:
  - 主题:bytedance-red(醒目)
  - 时间 / 地点 / 议程 / 报名方式表格化
  - 加粗报名按钮文字
```

## 9. 读书笔记

```
用户:这是我《xxx》读书笔记,排成公众号,极简白底

Skill:
  - 主题:minimal-cool
  - 金句用引用块 + 加粗
  - 章节用 H2
  - 末尾加"延伸阅读"
```

## 10. 换主题对比

```
用户:帮我把这段排 5 种不同主题,我挑一个发

Skill:
  - 渲染 5 个版本:classic-orange / minimal-black / focus-blue / gradient-purple / card-green
  - 每个版本返回预览 URL
  - 用户选中后推送草稿
```

## 常见反馈

- **"自动重写文案可能不贴脸"** → 提前给 Skill 详细 brief;或用"帮我排版这段(粘贴)"替代"帮我写"
- **"草稿推送失败"** → 检查 AppID+Secret 是否过期,或确认公众号是否已认证
- **"主题丑"** → 换一个;[theme-gallery](https://aipickgold.com/theme-gallery) 预览
